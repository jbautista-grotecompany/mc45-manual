    5 Recipe Setup¶

    5.1 Recipe Screen: Field Reference¶

    All production parameters are stored in the active recipe. Editable fields have a yellow background with green text. See Section 12: OI Reference for screen layout and Appendix A: Recipe Starting Points for starting values.

    Field Range Description
    RAKE SPEED 10–100 % RAKE motor speed.
    SPINNER SPEED 10–100 % SPINNER motor speed.
    PRODUCT CONV 0–60 FPM PRODUCT CONVEYOR speed.
    LINE RATE 0–48 PPM Target throughput in portions per minute. Used by the PLC to calculate PORTION CONVEYOR move speed.
    FEED-Z CONV 10–30 FPM FEED-Z CONVEYOR speed. (Option)
    FEED CONV 0–100 FPM FEED CONVEYOR speed. (Option)
    HOPPER LOW WEIGHT 0–40 lb PORTION CONVEYOR load cell lower threshold. FEED-Z CONVEYOR starts when weight falls below this value.
    HOPPER HIGH WEIGHT 0–40 lb PORTION CONVEYOR load cell upper threshold. FEED-Z CONVEYOR stops when weight exceeds this value.
    BUCKET OPEN TIME 0–1000 ms Time the BUCKET remains open on each deposit cycle.
    BUCKET FALL TIME 0–1000 ms Time allowed for topping to fall from the BUCKET to the target before the CARPET retracts.
    MASK OFFSET 0–14 in CARPET retracted position offset from home. Sets the starting position before each extend move.
    STROKE LENGTH 0–16 in CARPET extend distance from the retracted position.
    TARGET DELAY 0–3000 ms Time between TARGET SENSOR detection and CARPET trigger.
    EXTEND RATIO 0–2.0 Ratio of CARPET extend speed to retract speed.
    PORTION CONV MOVE LENGTH in Distance the PORTION CONVEYOR advances per machine cycle.
    PORTION CONV TIME 0–10000 ms Time window available for the PORTION CONVEYOR to complete its move.
    SKIP MODE ON / OFF When active, the Depositor deposits on every other target. See Section 5.3.

    5.2 Loading and Saving Recipes¶

    1. Navigate to the RECIPE MANAGEMENT screen on the OI.
    2. To load: enter the recipe number in RECIPE #. Press and hold LOAD RECIPE for three seconds.
    3. To create a new recipe: enter a recipe number and name. Enter all parameters. Press and hold SAVE RECIPE for three seconds.

    Note

    The Depositor stores up to 128 recipes. Saving to an existing recipe number overwrites it without confirmation. Verify the recipe number before saving.

    5.3 Skip Mode¶

    Skip Mode deposits topping on every other target. When active, the Depositor alternates between depositing and skipping each successive target.

    To enable Skip Mode, set the SKIP MODE field to ON in the active recipe and save. The SKIP MODE indicator on the OI illuminates when active.

    If the skip phase is out of sync with the line — depositing on targets that should be skipped — press SYNC on the OI. Each press shifts the skip phase by one target.

    5.4 Rake Height Setup¶

    RAKE HEIGHT is set manually using the hand-crank adjustment on the machine frame. The RAKE HEIGHT INDICATOR shows the current setting as a relative position number. This number has no engineering unit. It is a reference only.

    Start from the baseline value in Appendix A: Recipe Starting Points for the topping type. Adjust in small increments. Allow approximately 100 targets to pass through before evaluating the bed after each adjustment.

    Topping Compaction Assessment¶

    Before adjusting RAKE HEIGHT, assess the topping's compaction behavior:

    1. Collect a representative sample sufficient to fill one hand.
    2. Apply light pressure and open your hand. Poke the sample. If it separates freely, the topping is loose.
    3. Squeeze moderately and open your hand. Poke the sample. Does it fall apart or hold its shape?
    4. Squeeze firmly and open your hand. Try to break the sample apart. If shreds pull apart cleanly, compaction is not at the damage threshold. If the sample holds together and will not separate without tearing, that is the damage threshold. The bed must stay below this pressure level.
    5. Adjust RAKE HEIGHT in small increments. Allow approximately 100 targets to pass before evaluating.
    6. When the bed is uniform with no voids, save the recipe.

    Compaction Assessment

    High compaction (topping holds shape under light or moderate pressure): indicates high moisture or stickiness. Raise RAKE HEIGHT first to reduce tine engagement depth. A shallower tine bite reduces compressive force on the pile. Reduce HOPPER HIGH WEIGHT only if raising RAKE HEIGHT alone does not resolve compaction.

    Low compaction (topping crumbles freely under firm pressure): start from the Appendix A baseline values. If the bed has voids, raise RAKE HEIGHT in small increments.

    Both over-fill and under-fill produce the same symptom: voids and uneven coverage. If raising RAKE HEIGHT does not resolve voids, evaluate whether HOPPER HIGH WEIGHT is too high or too low before making further height adjustments.

    Typical HOPPER HIGH WEIGHT ranges:

    • Shredded cheese (standard): 15–20 lb.
    • IQF / frozen product: 12–18 lb.
    • Mixed or vegetable toppings with varied piece sizes: similar to cheese range, with RAKE SPEED reduced.

    Long or sticky shreds tend to bridge across the RAKE tines. The tines are staggered in two offset rows. A long shred can span the gap between tines and drag laterally as the RAKE turns. That dragging pulls material away and leaves a void in the bed. Raise RAKE HEIGHT to reduce tine engagement depth so shreds release rather than drag. Do not raise HOPPER HIGH WEIGHT to compensate: a heavier pile increases compaction pressure, which worsens the problem on sticky material. If voids persist after raising RAKE HEIGHT, reduce RAKE SPEED. RAKE SPEED should not exceed 60%.

    Fine or short shreds behave more freely. The pile can be lighter, and RAKE HEIGHT can run lower.

    Note

    Repeat the assessment when the topping lot changes, after extended idle periods, or when ambient conditions differ significantly from normal.
